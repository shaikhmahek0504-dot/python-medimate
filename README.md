# 💊 MediMate

MediMate is a simple and user-friendly medication reminder web application built using **Python** and **Flask**. It helps users organize and manage their medicine schedules by allowing them to add and view medication reminders through a clean web interface.

---

## 🚀 Features

- 💊 Add medication reminders
- 📋 View all saved reminders
- 💾 Store reminder data in a JSON file
- 🌐 Simple and responsive web interface
- ⚡ Fast and lightweight application

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JSON

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/shaikhmahek0504-dot/python-medimate.git
```

### 2. Go to the project directory

```bash
cd python-medimate
```

### 3. Install dependencies

```bash
pip install flask
```

### 4. Run the application

```bash
python app.py
```

### 5. Open in your browser

```
http://127.0.0.1:5000
```
